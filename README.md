# NEPHILIM v2.0

Cybersecurity signal awareness radar and network monitoring suite.

## Tech Stack

React • TypeScript • Convex • Vite • Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/    # Reusable UI components
├── pages/         # Page components
├── lib/           # Utility functions
└── App.tsx        # Main app component
convex/
├── schema.ts      # Database schema
└── *.ts           # Backend functions
```
